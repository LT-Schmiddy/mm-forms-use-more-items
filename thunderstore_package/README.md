# Forms Use More Items

Enables Link's transformation to use a few more items than they normally can.

Deku Link can now use:

* Deku Sticks
* Magic Beans

Goron Link can now use:

* Bombs
* Bombchus
* Deku Nuts
* Magic Beans

Zora Link can now use:

* Bombs
* Bombchus
* Deku Sticks
* Deku Nuts
* Magic Beans
